  Project 1: User Level Non-Preemptive Threads Library

This Threads library is in compliance with POSIX standard. The library provides APIs to create, join, yield and exit threads. The threads are light-weight threads created with the clone() system call. The special feature of this threading library is the FUTEX (Fast-User-Level-Mutex). The number of active light-weight processes is controlled with the FUTEX.
