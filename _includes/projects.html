Academic Projects

  Project 1: User Level Non-Preemptive Threads Library
This Threads library is in compliance with POSIX standard. The library provides APIs to create, join, yield and exit kernel threads. The threads are light-weight threads created with the clone() system call. The special feature of this threading library is the FUTEX (Fast-User-Level-Mutex). The number of active light-weight processes is controlled with the FUTEX.
  Project 2: Synchronization Primitives for User-Level Threading Library
This project provides APIs of the sysnchronization primitives - Mutexes, Conditional Variables and Barriers. These methods have the same signature as their corresponding counterparts in the POSIX threads standard with pthread_ replaced by mythread_. The lock implementation uses an early+late back-off test-and-test-and-set (TTS) lock with a threshold of 50 tries for busy-wait looks.
